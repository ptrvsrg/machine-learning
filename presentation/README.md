# Презентация: градиентный бустинг на деревьях решений

Slidev-презентация на 10–15 минут (XGBoost, LightGBM, CatBoost). Исходник слайдов: [`slides.md`](./slides.md).

## Запуск

```bash
npm install
npm run dev
```

С `pnpm`:

```bash
pnpm install
pnpm dev
```

Откройте <http://localhost:3030> (порт может отличаться — смотрите вывод в терминале).

## Сборка

```bash
npm run build
```

## Экспорт в PDF

Нужен установленный Chromium для Playwright (пакет `playwright-chromium` в `devDependencies`):

```bash
npm run export
```
